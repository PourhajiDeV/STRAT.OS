# ⚡ STRAT.OS ⚡

**STRAT.OS** یک سیستم برنامه‌ریزی استراتژیک روزانه و پلتفرم تحلیل بهره‌وری است که با هدف بهینه‌سازی عملکرد، مدیریت هوشمند زمان و تحلیل دقیقِ الگوهای رفتاری توسعه یافته است.

## ✨ ویژگی‌های کلیدی
* **داشبورد KPI پیشرفته:** مشاهده لحظه‌ای وضعیت اهداف، تسک‌های تکمیل شده و موارد شکست‌خورده.
* **هسته تحلیل رفتار (Analytics Engine):** * نمودار میله‌ای پویا برای بررسی فرکانس موفقیت هفتگی.
    * ماتریس تفکیک اولویت‌ها (High/Medium/Low) با نوار پیشرفت مجزا.
    * **Focus Score:** محاسبه شاخص تمرکز بر اساس ترکیبی از تسک‌های موفق و پایداری عادات.
    * **تشخیص هوشمند تیپ شخصیتی بهره‌وری:** تحلیل زمان‌های طلایی انجام کار در ۸ بازه دقیق (نیمه‌شب، بامداد، صبح زود، صبح، ظهر، بعدازظهر، عصر و شب).
* **زمان‌سنج و تمرکز:** مجهز به کرنومتر با قابلیت ثبت دور (Lap) و تایمر معکوس تمرکز به همراه هشدار صوتی و اعلان مرورگر.
* **تقویم و گزارش‌گیری:** ثبت گزارش‌های روزانه، محاسبه نرخ موفقیت هر روز و امکان یادداشت‌گذاری اختصاصی برای تواریخ گذشته.
* **پروفایل و کارت شناسایی سایبرنتیک:** محاسبه سطح (Level) و XP کاربر با قابلیت دانلود کارت شناسایی به صورت تصویر با کیفیت بالا.
* **سیستم مدیریت هوشمند:** قابلیت قفل‌گذاری روزانه (End-Day) برای بستن پرونده‌ی کاری هر روز.
* **بخش درباره STRAT.OS:** معرفی کامل سیستم، تیم توسعه و مشخصات فنی.
* **تنظیمات هسته:** امکان تغییر تایم‌زون، فرمت تقویم (شمسی/میلادی)، انتخاب ۱۸ پوسته کلی و ۸ پوسته کارت، و مدیریت دیتابیس (اتصال به SQLite محلی، MySQL یا LocalStorage).

## 🛠 فنی و معماری
* **تکنولوژی‌ها:** HTML5, CSS3, JavaScript (ES6+).
* **طراحی:** رابط کاربری مبتنی بر **Glassmorphism** و استفاده از فونت محلی **Mikhak-FD** (نمایش خودکار اعداد فارسی).
* **ذخیره‌سازی:** پشتیبانی از **Local Persistence** (مرورگر)، اتصال به فایل سیستم **SQLite** و API سرور **MySQL**.
* **توسعه:** پروژه‌ای از **PourhajiDeV** با همکاری **Gemini** (AI-Assisted) و تقدیم‌شده به **Haniyeh**.

## 🚀 راهنمای استفاده
برای اجرای پروژه کافیست فایل `index.html` را در یک مرورگر مدرن باز کنید. پوشه `fonts` شامل وزن‌های مختلف فونت میخک باید در کنار فایل اصلی قرار داشته باشد. تمامی اطلاعات شما به صورت امن ذخیره شده و در پایان هر روز می‌توانید از دکمه **"پایان روز / قفل"** برای مدیریت چرخه‌ی کاری استفاده کنید.
