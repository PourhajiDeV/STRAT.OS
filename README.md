# ⚡ STRAT.OS ⚡

**STRAT.OS** یک سیستم برنامه‌ریزی استراتژیک روزانه و پلتفرم تحلیل بهره‌وری است که با هدف بهینه‌سازی عملکرد، مدیریت هوشمند زمان و تحلیل دقیقِ الگوهای رفتاری توسعه یافته است.

## ✨ ویژگی‌های کلیدی
* **داشبورد KPI پیشرفته:** مشاهده لحظه‌ای وضعیت اهداف، تسک‌های تکمیل شده و موارد شکست‌خورده.
* **هسته تحلیل رفتار (Analytics Engine):** * نمودار میله‌ای پویا برای بررسی فرکانس موفقیت هفتگی.
    * ماتریس تفکیک اولویت‌ها (High/Medium/Low) با نوار پیشرفت مجزا.
    * **Focus Score:** محاسبه شاخص تمرکز بر اساس ترکیبی از تسک‌های موفق و پایداری عادات.
    * **تشخیص تیپ شخصیتی بهره‌وری:** تحلیل زمان‌های طلایی (سحرخیز/شب‌زنده‌دار/نیمروزی).
* **سیستم مدیریت هوشمند:** قابلیت قفل‌گذاری روزانه (End-Day) برای بستن پرونده‌ی کاری هر روز.
* **تنظیمات هسته:** امکان تغییر تایم‌زون، فرمت تقویم (شمسی/میلادی) و مدیریت دیتابیس (Factory Reset).

## 🛠 فنی و معماری
* **تکنولوژی‌ها:** HTML5, CSS3, JavaScript (ES6+).
* **طراحی:** رابط کاربری مبتنی بر **Glassmorphism** با استفاده از فونت **Vazirmatn**.
* **ذخیره‌سازی:** سیستم **Local Persistence** برای حفظ یکپارچگی داده‌ها در مرورگر.
* **توسعه:** پروژه‌ای از **PourhajiDeV** با همکاری **Gemini** (AI-Assisted).

## 🚀 راهنمای استفاده
برای اجرای پروژه کافیست فایل `index.html` را در یک مرورگر مدرن باز کنید. تمامی اطلاعات شما در قالب یک فایل ایزوله در مرورگر ذخیره شده و به محض باز کردن برنامه، در دسترس خواهد بود. در پایان هر روز می‌توانید از دکمه **"پایان روز / قفل"** برای بازنشانی سیستم جهت شروع روز بعد استفاده کنید.
