# ⚡ STRAT.OS

**STRAT.OS** یک سیستم برنامه‌ریزی استراتژیک روزانه است که با هدف افزایش بهره‌وری و مدیریت هوشمند وظایف طراحی شده است. این پروژه بر پایه‌ی اصول طراحی مدرن و رابط کاربری مینیمال توسعه یافته تا فضایی متمرکز برای مدیریت تسک‌ها، پیگیری عادات و ثبت ایده‌های روزانه فراهم کند.

## 🚀 ویژگی‌های کلیدی
* **داشبورد مدیریتی:** مشاهده وضعیت عملکرد روزانه از طریق KPIهای سیستمی.
* **سیستم مدیریت وظایف:** قابلیت تعریف و اولویت‌بندی تسک‌ها (High, Medium, Low).
* **مانیتور پویای عادات:** پیگیری پیشرفت عادات با قابلیت تنظیم اهداف شخصی‌سازی شده.
* **تحلیل بصری:** سیستم آنالیز فرکانس برای بررسی میزان موفقیت در طول هفته.
* **ذخیره‌سازی لوکال:** امنیت کامل داده‌ها در محیط مرورگر بدون نیاز به زیرساخت‌های جانبی.

## 🛠 توسعه و معماری
این پروژه حاصل فرآیند طراحی و توسعه توسط **PourhajiDeV** می‌باشد. در بخش پیاده‌سازی منطق برنامه و بهینه‌سازی کد، از راهکارهای **AI-Assisted** (توسعه‌یافته با همکاری **Gemini**) استفاده شده است تا دقت و کارایی هسته‌ی سیستم به حداکثر برسد.
* **تکنولوژی‌ها:** HTML5, CSS3, JavaScript (ES6+).
* **طراحی:** استفاده از تکنیک Glassmorphism و فونت اختصاصی Vazirmatn.

## 📝 نحوه استفاده
برای اجرای پروژه کافیست فایل `index.html` را در یک مرورگر مدرن باز کنید. تمامی اطلاعات شما در قالب یک فایل ایزوله در مرورگر ذخیره شده و به محض باز کردن برنامه، در دسترس خواهد بود.
