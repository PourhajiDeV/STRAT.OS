# ⚡ STRAT.OS - سیستم برنامه‌ریزی استراتژیک روزانه

**STRAT.OS** یک ابزار مدرن، مینیمال و مبتنی بر وب برای مدیریت وظایف (Task Management)، پیگیری عادات (Habit Tracking) و یادداشت‌برداری سریع است. این پروژه با تمرکز بر تجربه کاربری (UX) و طراحی بصری شیشه‌ای (Glassmorphism) توسعه یافته تا بهره‌وری شما را در محیطی آرام و متمرکز افزایش دهد.

## 🚀 ویژگی‌های کلیدی
- **داشبورد هوشمند:** مشاهده وضعیت لحظه‌ای راندمان روزانه و KPIهای کاری.
- **مدیریت وظایف:** قابلیت تعریف تسک‌ها با تعیین اولویت (High, Medium, Low).
- **مانیتور پویای عادات:** پیگیری پیشرفت عادات روزانه با قابلیت افزایش و کاهش مقدار.
- **تحلیل داده‌ها:** سیستم نمودار بصری برای بررسی نرخ موفقیت در طول هفته.
- **ذخیره‌سازی محلی (Local Storage):** تمامی اطلاعات شما در مرورگر ذخیره شده و نیازی به دیتابیس خارجی یا ثبت‌نام ندارد؛ حریم خصوصی ۱۰۰٪.
- **رابط کاربری مدرن:** طراحی واکنش‌گرا (Responsive) با استفاده از المان‌های Glassmorphism و انیمیشن‌های نرم.

## 🛠 تکنولوژی‌های استفاده شده
این پروژه با رویکرد "Vanilla Stack" توسعه یافته است:
* **HTML5:** ساختار معنایی صفحه.
* **CSS3:** طراحی بصری، انیمیشن‌های keyframes و تکنیک‌های Glassmorphism.
* **JavaScript (ES6):** منطق برنامه، مدیریت وضعیت (State Management) و ارتباط با `localStorage`.
* **فونت:** استفاده از فونت بهینه **Vazirmatn**.
